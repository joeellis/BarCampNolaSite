So What Is Barcamp Nola?

BarCampNola is a local incarnation of the international network of user generated conferences called BarCamps. BarCamp is an ad-hoc gathering born from the desire for people to share and learn in an open environment. It is an intense event with discussions, demos, and interaction from attendees. It is an open event where each attendee is encouraged to participate and present.
